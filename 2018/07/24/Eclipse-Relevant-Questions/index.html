Eclipse相关问题整理

修改Eclipse Git插件中默认的作者和提交者

  • 打开 Windows -> Preferences -> Team -> Git -> Configuration
  • 修改User Settings，点击Add Entry
  • 添加两个键值对分别为:
    • user.name xxxx
    • user.email xxxxx@xxx.com
  • 这时在Git Staging中的Commit message栏位中出现的Author和Committer就是你想要的提交者了。
